# CommentRaid - Idol On Stage v1.0.1

CommentRaid - Idol On Stage は、[CommentRaid - Base]をベースにした、ライブ配信を盛り上げるための「応援・育成型」プラグイン（ルールパック）です。

従来のレイドバトル（ボスを倒す）とは異なり、視聴者のコメントを「応援（Cheer）」として蓄積し、配信者の「ボルテージ（VOLTAGE）」を上げてレベルアップを目指す、ポジティブなゲームモードを提供します。

## 🌟 特徴

- **応援システム:** 視聴者がコメントやギフトを送ることでボルテージが上昇。
- **レベルアップ:** ボルテージが最大になるとレベルアップ。アイドルとしての称号が進化します。
- **配信者スキル:** 配信者自身がコマンド（!hypetime等）を打つことで、一定時間応援力を倍増させるバフを発動可能。
- **リッチな演出:** ハートが飛び交うエフェクト、レベルアップ時のスタンプ演出、斜めに走査する三角形のサイバーなゲージUI。
- **簡単設定:** 専用の `config_editor.html` を使い、ブラウザ上で直感的に設定値を変更・保存できます。

## 📦 フォルダ構成

```text
CRB_idolonstage_v1/
├── data/
│   ├── config.js          # 基本システム設定
│   └── idol_config.js     # アイドルの名前、難易度、UI設定
├── plugins/
│   ├── data/
│   │   ├── cheer_table.js      # 応援アクション（!clap等）のリスト
│   │   ├── idol_profile.js     # アイドル称号（二つ名・役職）の定義
│   │   └── idol_skill_data.js  # 配信者スキルの性能定義
│   ├── idol_logic.js      # ゲーム進行・レベル計算ロジック
│   ├── cheer_plugin.js    # 応援値計算・バフ適用処理
│   ├── idol_fx.js         # ハートエフェクト拡張
│   ├── idol_skill.js      # 配信者スキル発動処理
│   └── ui.html / style.css # 専用UIコンポーネント
├── index.html             # メインエントリ
└── config_editor.html      # 設定エディタ
```

## 🚀 使い方

1.  **インストール:** `CRB_idolonstage_v1` フォルダをわんコメの `templates/custom/` に配置します。
2.  **基本設定:** `config_editor.html` をブラウザで開き、配信者の名前や難易度を設定してファイルを保存（上書き）します。
3.  **起動:** わんコメのテンプレート設定から `CRB_idolonstage_v1/index.html` を選択して読み込みます。
4.  **配信:** 視聴者がコメントすると応援が始まります。配信者は `!hypetime` などで盛り上げを加速させましょう。

## 🛠 カスタマイズ

-   **応援アクション:** `plugins/data/cheer_table.js` で、特定のコマンドに対する倍率を変更できます。
-   **スキル性能:** `plugins/data/idol_skill_data.js` で、スキルのリキャスト時間や倍率、効果時間を自由に変更可能です。
-   **称号リスト:** `plugins/data/idol_profile.js` で、進化する名前のバリエーションを増やせます。

## 権利関係・カスタマイズに関する備考

- 本テンプレートは、配信者様の活動方針やリスナー層の傾向に合わせて、柔軟に素材を差し替えられるよう設計されています。

### 素材の自由な差し替えについて
- デフォルトで含まれている画像素材（スタンプ等）は、以下の手順で**配信者様ご自身のオリジナル素材や、許諾を得たイラスト**に差し替えることが可能です。
- 同梱画像素材を使用しない場合は、該当するスタンプ演出のみ非表示になります。テンプレート自体はそのまま動作します。

* **オリジナルイラストの利用**: 
  `idol_config.js` 内の `STAMP_START_URL` や `STAMP_DEFEAT_URL` のパスを、ご自身で用意したPNGファイル等に書き換えるだけで適用されます。
* **「非AI素材」への限定**: 
  画像生成AIを避けたい場合、手描きイラストやフリー素材サイト（規約に準じたもの）の画像を使用することで、素材の出自を配信者様の方針に合わせて管理することができます
